# Repo Manage Index

This subtree contains the repository-management documentation: plans, architecture notes, status tracking, task files, and handoffs.

The wider `docs/` directory now also contains non-management reference material such as the report PDFs, presentation files, and figure assets.

## Working Interpretation

- The repository root is the umbrella project.
- Active code lives under `src/`.
- The main retained notebook lives under `notebooks/main/`.
- Historical scripts, notebooks, MATLAB files, and reports are separated into `archive/`, `notebooks/archive/`, and `artifacts/`.
